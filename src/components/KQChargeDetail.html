Project Overview

KQCharge is a comprehensive mobile application designed for electric vehicle (EV) owners and charging station operators. The app provides real-time monitoring of charging stations, seamless payment integration, and efficient fleet management capabilities.

Built with Flutter and GetX state management, the application offers a smooth user experience across both iOS and Android platforms, featuring intuitive navigation and responsive design.
